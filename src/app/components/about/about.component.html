This Angular project uses the Angular Router to navigate between different components like home and about. The HTTP client is also used to fetch data from a REST API. This HTTP client is injected into the service that fetches the data and therefore it is used too in the component that renders the data. Angular dependency injection is used to inject the HTTP client service into the component. The views are rendered using the NgIf directive, now known as @If. The Angular CLI is used to build and test this project.
